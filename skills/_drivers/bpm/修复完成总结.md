## XPDL 格式规范

### 开始节点格式
```
ParticipantID;FirstActivityID;X;Y;Routing

示例：
Participant_Start;act_start;50;200;NO_ROUTING
```

### 结束节点格式（支持多个）
```
ParticipantID;LastActivityID;X;Y;Routing|ParticipantID;LastActivityID;X;Y;Routing

示例：
Participant_End1;act_end1;800;100;NO_ROUTING|Participant_End2;act_end2;800;200;NO_ROUTING
```

### Listeners XML格式
```xml
<itjds:Listeners>
  <itjds:Listener Id="listener_001" Name="流程启动监听" 
                  ListenerEvent="PROCESS_START" 
                  RealizeClass="com.example.StartListener"/>
</itjds:Listeners>
```

### RightGroups XML格式
```xml
<itjds:RightGroups>
  <itjds:RightGroup Id="rg_001" Name="默认组" Code="DEFAULT" 
                    Order="1" DefaultGroup="YES"/>
</itjds:RightGroups>
```

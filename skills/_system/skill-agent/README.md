# skill-agent

Agent管理服务 - 提供Agent注册、监控、拓扑管理、聊天、会话管理等完整功能

## 功能特性

- Agent注册 - 注册新Agent
- 心跳检测 - Agent心跳检测
- 监控指标 - 获取Agent监控指标
- 拓扑管理 - 获取Agent拓扑结构
- 聊天交互 - Agent聊天交互功能
- 会话管理 - Agent会话管理功能
- 消息服务 - Agent消息队列服务
- A2A通信 - Agent间通信协议

## 快速开始

### 安装

```bash
skill install skill-agent
```

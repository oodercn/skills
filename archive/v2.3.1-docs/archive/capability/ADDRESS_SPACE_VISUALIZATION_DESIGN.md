# Skills能力地址可视化框架设计

## 一、地址空间概览

### 1.1 17种能力分类

| 序号 | 代码 | 名称 | 地址范围 | 基地址 | 描述 |
|------|------|------|----------|--------|------|
| 1 | sys | 系统核心 | 0x00-0x07 | 0x00 | 系统注册、配置、能力管理 |
| 2 | org | 组织服务 | 0x08-0x0F | 0x08 | 钉钉、飞书、企业微信、LDAP |
| 3 | auth | 认证服务 | 0x10-0x17 | 0x10 | 用户认证、令牌管理 |
| 4 | net | 网络服务 | 0x18-0x1F | 0x18 | 代理、DNS |
| 5 | vfs | 文件存储 | 0x20-0x27 | 0x20 | 本地、MinIO、OSS、S3 |
| 6 | db | 数据库 | 0x28-0x2F | 0x28 | MySQL、PostgreSQL、MongoDB、Redis |
| 7 | llm | 大语言模型 | 0x30-0x37 | 0x30 | Ollama、OpenAI、通义、DeepSeek |
| 8 | know | 知识库 | 0x38-0x3F | 0x38 | 向量库、RAG、嵌入服务 |
| 9 | payment | 支付服务 | 0x40-0x47 | 0x40 | 支付宝、微信支付、银联 |

| 12 | mon | 监控服务 | 0x58-0x5F | 0x58 | 健康检查、指标、日志 |
| 13 | iot | 物联网 | 0x60-0x67 | 0x60 | K8s、托管、OpenWrt |
| 14 | search | 搜索服务 | 0x68-0x6F | 0x68 | Elasticsearch |
| 15 | sched | 调度服务 | 0x70-0x77 | 0x70 | Quartz调度 |
| 16 | sec | 安全服务 | 0x78-0x7F | 0x78 | 访问控制、审计 |
| 17 | util | 工具服务 | 0xF0-0xFF | 0xF0 | 报表、分享、市场 |

## 三、前端实现方案

### 3.1 API接口

```javascript
// 获取地址空间配置
GET /api/v1/config/addresses

// 获取分类配置
GET /api/v1/config/categories

// 获取能力状态
GET /api/v1/capabilities/status

// 获取Fallback链路
GET /api/v1/capabilities/{address}/fallback-chain
```

### 3.2 数据结构

```typescript
interface AddressCategory {
  name: string;
  displayName: string;
  range: string;
  baseAddress: number;
  color: string;
  icon: string;
}

interface CapabilityAddress {
  address: number;
  name: string;
  category: string;
  description: string;
  fallback?: number;
  status: 'active' | 'inactive' | 'pending';
  provider?: string;
  version?: string;
}

interface FallbackChain {
  from: number;
  to: number | null;
  condition: 'on_failure' | 'required' | 'optional';
}
```

## 五、交互设计

### 5.1 地址地图交互

1. **悬停**: 显示地址详情tooltip
2. **点击**: 打开能力详情弹窗
3. **拖拽**: 平移地图
4. **滚轮**: 缩放地图
5. **筛选**: 按分类筛选显示

### 5.2 分类树交互

1. **展开/收起**: 点击节点展开子节点
2. **搜索**: 输入关键字搜索
3. **拖拽排序**: 自定义排序
4. **右键菜单**: 添加/编辑/删除能力

### 5.3 能力卡片交互

1. **状态刷新**: 实时更新状态
2. **测试连接**: 测试能力可用性
3. **配置编辑**: 编辑能力配置
4. **查看日志**: 查看能力运行日志

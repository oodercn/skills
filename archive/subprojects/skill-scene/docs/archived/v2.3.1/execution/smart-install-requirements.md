## 一、概述

### 1.1 背景

场景技能的安装是一个复杂的过程，涉及依赖检查、参数配置、权限分配、菜单注册等多个环节。为了降低用户使用门槛，需要提供两种安装方式：

1. **LLM协助安装** - 通过AI助手引导用户完成安装
2. **手工安装** - 用户通过向导界面手动完成安装

### 1.2 目标用户

| 用户类型 | 说明 | 推荐安装方式 |
|----------|------|--------------|
| **普通用户** | 非技术背景，不熟悉系统配置 | LLM协助安装 |
| **高级用户** | 技术背景，熟悉系统配置 | 手工安装 |
| **管理员** | 系统管理员，批量安装 | 手工安装 + 批量操作 |

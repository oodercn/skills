# Skills基础驱动配置与依赖关系分析

## 一、基础驱动配置总览

### 1.1 系统安装时必需的基础配置

| 序号 | 配置项 | 地址 | 分类 | 必需性 | 说明 |
|------|--------|------|------|--------|------|
| 1 | 系统注册中心 | 0x00 | sys | **必需** | 系统启动时第一个注册 |
| 2 | 能力管理服务 | 0x02 | sys | **必需** | 管理所有能力 |
| 3 | 协议处理服务 | 0x03 | sys | **必需** | 处理协议转换 |
| 4 | 用户认证服务 | 0x10 | auth | **必需** | 用户登录认证 |
| 5 | 令牌管理服务 | 0x11 | auth | **必需** | Token管理 |
| 6 | VFS基础服务 | 0x20 | vfs | **必需** | 文件系统基础 |
| 7 | 本地文件存储 | 0x21 | vfs | 推荐 | 默认存储方案 |
| 8 | 监控基础服务 | 0x58 | mon | **必需** | 系统监控 |
| 9 | 健康检查服务 | 0x59 | mon | **必需** | 健康状态检测 |

### 1.2 可选的基础驱动配置

| 分类 | 配置项 | 地址 | 说明 | 选择场景 |
|------|--------|------|------|----------|
| **数据库** | MySQL | 0x28 | 关系型数据库 | 需要关系数据存储 |

## 三、安装向导中的驱动选择配置

### 3.1 系统安装时需要选择的驱动

| 步骤 | 配置项 | 默认值 | 可选项 | 说明 |
|------|--------|--------|--------|------|
| 1 | 数据库选择 | MySQL | MySQL, PostgreSQL, MongoDB | 主数据库 |
| 2 | 缓存选择 | Redis | Redis, Memcached | 缓存服务 |
| 3 | 存储选择 | 本地存储 | 本地, MinIO, OSS, S3 | 文件存储 |
| 4 | LLM选择 | DeepSeek | Ollama, OpenAI, 通义, DeepSeek, 豆包 | AI服务 |
| 5 | 组织选择 | 本地组织 | 本地, 钉钉, 飞书, 企微, LDAP | 用户管理 |

### 3.2 场景安装时需要选择的驱动

| 场景类型 | 需要的驱动配置 | 配置来源 |
|----------|----------------|----------|
| **LLM对话场景** | LLM Provider选择 | 安装向导步骤6 |
| **知识问答场景** | LLM + 知识库 + RAG | 安装向导步骤6 |
| **协作场景** | 消息服务 + 通知服务 | 系统默认配置 |
| **入职助手场景** | LLM + 知识库 + 组织服务 | 安装向导步骤6 |

### 5.2 获取依赖关系

```http
GET /api/v1/config/dependencies
```

**响应示例**:
```json
{
  "status": "success",
  "data": {
    "skills": [
      {
        "id": "skill-knowledge-qa",
        "name": "知识问答场景能力",
        "dependencies": ["skill-knowledge-base", "skill-rag"],
        "requiredAddresses": ["0x38", "0x3A", "0x30"]
      }
    ],
    "addresses": [
      {
        "address": "0x30",
        "name": "LLM_BASE",
        "fallback": "0x31",
        "usedBy": ["skill-llm-chat", "skill-knowledge-qa"]
      }
    ]
  }
}
```

### 5.3 保存驱动配置

```http
POST /api/v1/config/drivers
```

**请求体**:
```json
{
  "llm": {
    "selected": "skill-llm-deepseek",
    "config": {
      "apiKey": "sk-xxx",
      "model": "deepseek-chat"
    }
  },
  "db": {
    "selected": "skill-db-mysql",
    "config": {
      "host": "localhost",
      "port": 3306,
      "database": "ooder_skills"
    }
  }

```

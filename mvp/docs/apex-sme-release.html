🎯 为什么选择 Apex？

专为小微企业设计的 AI Agent 基础设施

1

轻量级零配置安装

无需复杂配置，一行命令即可启动。内置四级 Profile，从 micro（128MB）到 enterprise（1GB），按需选择，灵活部署。

2

5 秒极速启动

基于 Java 21 + Spring 3.x，结合 GraalVM 优化，micro 模式下仅需 5 秒即可完成启动，即刻投入使用。

3

场景驱动模型

以业务场景为中心，而非技术 API。场景 = 参与者 + 能力 + 知识库 + LLM，让 AI 真正理解业务语境。

4

永久记忆大模型

LLM + 知识资料库，打造企业专属的永久记忆。支持向量嵌入、RAG 检索增强，让 AI 记住企业知识。

5

多模型介入管控

支持 OpenAI、DeepSeek、千问、Ollama 等多模型。分部门统计用量，按需管控成本。

6

PageAgent 深度嵌入

Agent 可深入嵌入业务应用，拟人化角色设计，让 AI 成为团队的一员，而非孤立工具。

7

开源 Skills 库

内置 40+ 开源技能，覆盖问答、报表、通知等常见场景。技能热插拔，按需扩展。

8

四级用量统计

公司 → 部门 → 用户 → 模块，四级用量统计，精准掌控每一笔 AI 开支。

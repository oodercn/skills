    接口表述配置
    配置LLM可调用的知识库相关接口，包括接口描述、参数说明、返回格式等，帮助LLM正确使用知识库功能。

    知识库接口定义

    POST /api/v1/knowledge-bases/{kbId}/search
    知识检索接口

    接口描述

    在指定知识库中进行语义检索，返回与查询最相关的文档片段。

    请求参数

    参数名 类型 必填 说明
    query string 是 检索查询文本
    topK integer 否 返回结果数量，默认5
    threshold float 否 相似度阈值，默认0.7

    LLM调用提示

    当用户询问与{知识库名称}相关的问题时，使用此接口检索相关文档。
    GET /api/v1/knowledge-bases
    获取知识库列表

    接口描述

    获取当前用户可访问的所有知识库列表。

    LLM调用提示

    当需要了解有哪些知识库可用时，调用此接口获取列表。
    POST /api/v1/scene-groups/{sceneGroupId}/knowledge/search
    场景知识检索

    接口描述

    在场景绑定的所有知识库中进行跨层检索，按优先级返回结果。

    请求参数

    参数名 类型 必填 说明
    query string 是 检索查询文本
    layers array 否 指定检索层级：SCENE/PROFESSIONAL/GENERAL
    topK integer 否 返回结果数量，默认5

    LLM调用提示

    在场景对话中，使用此接口检索与场景相关的知识。检索会按场景层→专业层→通用层的优先级进行。

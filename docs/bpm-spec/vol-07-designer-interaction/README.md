## 1. 工具栏设计

### 1.1 主工具栏 (Toolbar)

主工具栏位于画布上方，提供常用操作按钮。

| 按钮组 | 按钮 | 图标 | 快捷键 | 说明 |
|--------|------|------|--------|------|
| 编辑 | 撤销 | undo | Ctrl+Z | 撤销上一步操作 |
| 编辑 | 重做 | redo | Ctrl+Y | 重做上一步操作 |
| 视图 | 放大 | zoomin | - | 放大画布 |
| 视图 | 缩小 | zoomout | - | 缩小画布 |
| 视图 | 适应 | grid | - | 适应画布大小 |
| 视图 | 缩放比例 | - | - | 显示当前缩放比例 |
| 对齐 | 左对齐 | align-left | - | 左对齐选中节点 |
| 对齐 | 居中对齐 | align-center | - | 居中对齐选中节点 |
| 对齐 | 右对齐 | align-right | - | 右对齐选中节点 |
| 操作 | 删除 | delete | Delete | 删除选中元素 |

### 1.2 头部工具栏 (Header)

头部工具栏位于页面顶部，提供文件操作按钮。

| 按钮 | 图标 | 快捷键 | 说明 |
|------|------|--------|------|
| 切换主题 | moon | - | 切换浅色/深色主题 |
| 导入 | upload | - | 导入YAML格式流程 |
| 导出 | download | - | 导出YAML格式流程 |
| 保存 | save | Ctrl+S | 保存当前流程 |

## 5. TAB页设计

### 5.1 TAB页功能

| 功能 | 说明 |
|------|------|
| 多流程编辑 | 支持同时打开多个流程 |
| TAB切换 | 点击TAB切换流程 |
| 关闭TAB | 点击关闭按钮关闭TAB |
| 脏标记 | 未保存的流程显示标记 |
| 恢复上次 | 刷新后恢复上次打开的流程 |

### 5.2 TAB页状态

| 状态 | 样式 | 说明 |
|------|------|------|
| 激活 | 高亮显示 | 当前编辑的流程 |
| 普通 | 默认样式 | 非激活的TAB |
| 脏标记 | 小圆点 | 有未保存更改 |

## 6. 版本管理功能

### 6.1 新建版本

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 右键流程节点 | 弹出右键菜单 |
| 2 | 点击"新建版本" | 创建新版本 |
| 3 | 输入版本说明 | 可选 |
| 4 | 确认创建 | 基于当前版本创建新版本 |

### 6.2 版本状态操作

| 操作 | 说明 |
|------|------|
| 激活版本 | 将版本设为激活状态，可用于启动流程实例 |
| 冻结版本 | 冻结版本，不可启动新实例 |
| 删除版本 | 删除指定版本（需确认） |
